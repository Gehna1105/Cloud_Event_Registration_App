# College Event Registration System

A simple Flask + SQLite based web application for college event registration.

## Features
- Student event registration form
- Stylish user interface
- Stores registrations using SQLite
- View all registrations page for admin

## Technologies Used
- Python
- Flask
- SQLite
- HTML + CSS

## Project Structure
College_Event_Registration/
│
├── Local-App/
│   ├── app.py
│   ├── requirements.txt
│
└── README.md

## How to Run Locally
1. Open terminal inside `Local-App`
2. Install requirements:
   pip install -r requirements.txt
3. Run the app:
   python app.py
4. Open browser:
   http://127.0.0.1:5000

## Routes
- `/` → Registration Form
- `/view` → View All Registrations
